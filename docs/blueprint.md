# **App Name**: eMall Portal

## Core Features:

- Owner Authentication: Secure login for property owners using JWT and bcrypt for password hashing.
- Tenant Authentication: Secure login for tenants using JWT. Tenant accounts are created by the property owner.
- Dashboard Summaries: At a glance summaries of key data and notifications on both Owner and Tenant dashboards.
- Rent Prediction: The system will tool be able to automatically suggest competitive market rental rates for a particular type of property in a particular area. It will also consider other factors to fine tune the recommendation.
- Complaint Submission: Tenants can submit complaints related to property issues, tracked for resolution by owners.
- Rent and Bill History: Comprehensive tracking of rent and bill payments for both owners and tenants.
- Profile Settings: User profile updates including contact info and password changes.

## Style Guidelines:

- Primary color: Subtle blue (#A0D2EB) to create a calm and trustworthy atmosphere.
- Background color: Light gray (#F5F5F5) for a clean and modern Apple-inspired look.
- Accent color: Soft orange (#FFB347) for key actions and highlights, providing warmth and friendliness.
- Body and headline font: 'Inter', sans-serif, providing a clean and modern look suitable for both headlines and body text.
- Use a consistent set of minimalist icons that match the Apple aesthetic.
- Design a clean and intuitive layout inspired by Apple's design principles, ensuring a premium and smooth user experience.
- Incorporate subtle animations (using Framer Motion) for UI elements and transitions to enhance interactivity and smoothness.
